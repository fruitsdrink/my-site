基于 [Tauri 2](https://v2.tauri.app/) + React + TypeScript 的桌面专注计时器。窗口无边框、可透明，可常驻系统托盘；支持默认主题与翻页时钟主题，明暗模式与窗口位置会持久化。

## 功能

- **计时器**：开始 / 暂停 / 继续 / 重置；完成后播放提示音
- **主题**：默认主题、翻页时钟；托盘菜单切换，选择持久化
- **外观**：明暗模式与窗口位置记忆
- **窗口**：无边框、可透明；关闭后隐藏到后台；可选置顶
- **托盘**：系统托盘菜单；macOS 下可隐藏 Dock / 任务栏图标

## 下载

预编译安装包见 [GitHub Releases](https://github.com/fruitsdrink/chronovia-desk/releases/latest)。当前提供 macOS Apple Silicon (`.dmg`)：下载后拖入「应用程序」即可。若遇 Gatekeeper 拦截，可在「系统设置 → 隐私与安全性」中允许打开。

## 本地开发

需要 [Bun](https://bun.sh/)、[Rust](https://www.rust-lang.org/tools/install) 及各平台 [Tauri 前置依赖](https://v2.tauri.app/start/prerequisites/)。

```bash
bun install
bun tauri dev
```

完整说明见仓库 [README](https://github.com/fruitsdrink/chronovia-desk#readme)。
